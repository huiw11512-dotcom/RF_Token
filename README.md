# RF Token HFSS Dataset

This repository contains 300 HFSS AEDT project files.

## Contents

- `hfss_aedt_300/`: 300 HFSS project files
- File naming: `sample_000000.aedt` through `sample_000299.aedt`

## Notes

These files are ANSYS HFSS AEDT project data files prepared for RF token antenna dataset work.
